# 中国风关卡设计大全

## 🎭 主角设定

**徐霞客** - 明代旅行家
- **背景**: 立志游历中华大地，记录山川地理
- **技能**: 
  - 轻功（跳跃、攀爬）
  - 诗词歌赋（文化类关卡）
  - 武艺防身（战斗类关卡）
  - 医术（治疗、解毒）

---

## 🏞️ 关卡类型总览（12 种）

### 一、Godot 横版动作类（4 种）

#### 1. 园林跑酷（Speed Run）
**地域特色**: 苏州园林
- **场景**: 亭台楼阁、假山池塘、月洞门、回廊
- **玩法**: 在古典建筑间快速穿梭
- **障碍**: 
  - 移动的月亮门
  - 翻转的石桌
  - 喷水的龙首雕塑
- **收集品**: 诗词碎片、印章
- **时间**: 60 秒

#### 2. 古塔战斗（Castle Battle）
**地域特色**: 杭州雷峰塔、西安大雁塔
- **场景**: 多层古塔内部
- **玩法**: 逐层挑战守塔者
- **敌人**:
  - 第 1-2 层：普通侍卫（2 个）
  - 第 3-4 层：精英武士 + 弓箭手（3 个）
  - 第 5 层：塔主（Boss）
- **机制**: 使用长枪、剑法攻击
- **波次**: 5 波

#### 3. 登山攀岩（Platform Challenge）
**地域特色**: 黄山、华山、泰山
- **场景**: 险峻山峰，云海缭绕
- **玩法**: 垂直向上攀爬
- **平台类型**:
  - 固定岩石（40%）
  - 移动云雾（20%）
  - 消失的台阶（20%）
  - 弹跳松枝（10%）
  - 传送瀑布（10%）
- **高度**: 2000 像素
- **失败**: 掉落深渊

#### 4. 神兽 Boss 战（Boss Battle）
**地域特色**: 各地传说神兽
- **Boss 列表**:
  1. **苏州**: 辟邪（园林守护兽）
  2. **杭州**: 白蛇（雷峰塔下）
  3. **黄山**: 麒麟（云海之巅）
  4. **西安**: 秦始皇陵兵马俑将军
  5. **西藏**: 雪狮（布达拉宫守护）

**白蛇 Boss 技能**:
- 阶段 1（100%-70%）:
  - 水漫金山（全屏水波）
  - 蛇尾扫击（范围攻击）
  - 召唤小青（小怪）
- 阶段 2（70%-30%）:
  - 雄黄弱点（周期性暴露弱点）
  - 法术攻击（追踪法术球）
- 阶段 3（30%-0%）:
  - 现出原形（体型变大，攻击范围增加）
  - 狂暴（攻击频率翻倍）

### 二、文化策略类（8 种）

#### 5. 园林题诗（Poetry）
**地域特色**: 苏州拙政园
- **玩法**: 根据园林景色创作/对诗
- **LLM 交互**: AI 扮演园主出题
- **题目示例**:
  - "请为拙政园的荷花池题诗一首"
  - "对出下联：'小桥流水人家'"
- **评分**: LLM 根据意境、格律评分
- **奖励**: 文人墨客声望

#### 6. 断桥邂逅（Dating）
**地域特色**: 杭州西湖断桥
- **剧情**: 遇见白娘子，展开浪漫故事
- **选项**:
  - 借伞（好感度 +10）
  - 吟诗（好感度 +5）
  - 直接表白（好感度 -5，太唐突）
- **多结局**:
  - 好感度>80: 成就"许仙第二"
  - 好感度 50-80: 成为好友
  - 好感度<50: 擦肩而过

#### 7. 长安诗会（AI Debate）
**地域特色**: 西安大唐芙蓉园
- **对手**: AI 李白、杜甫、白居易
- **玩法**: 诗词辩论
- **流程**:
  1. 抽牌决定先手
  2. 出牌 + 念诗（如出"月"牌，念"床前明月光"）
  3. AI 跟牌并反驳
  4. 裁判（AI 王维）评分
- **胜利**: 累计 3 轮高分

#### 8. 徽州商帮（Spy - 谁是卧底）
**地域特色**: 安徽徽州古村落
- **背景**: 徽商聚会，有人暗中破坏生意
- **玩家**: 6 个徽商之一
- **卧底**: 1-2 人（其他商帮派来）
- **流程**:
  1. 每人描述自己的商品（茶叶、丝绸、瓷器）
  2. 投票选出可疑者
  3. 卧底目标：不被发现
  4. 商人目标：找出卧底
- **特色**: 融入徽商文化（晋商、粤商竞争）

#### 9. 大唐牌坊（Card - 比大小）
**地域特色**: 西安大唐西市
- **对手**: 胡商（波斯商人）
- **玩法**: 扑克牌比大小 + 唐朝文化
- **牌面改造**:
  - A → 元宝
  - K → 状元
  - Q → 贵妃
  - J → 将军
  - 10-2 → 铜钱
- **魔法指令**:
  - "开元通宝"（交换一张牌）
  - "丝绸之路"（从对方偷一张牌）
  - "贞观之治"（重置所有牌）

#### 10. 云海幻境（Reverse - 反转世界）
**地域特色**: 黄山云海
- **机制**: 左右/上下反转
- **玩法**: 
  - 正常世界：向左走
  - 反转世界：向右走
  - 重力反转：跳=下落
- **关卡设计**:
  - 需要在两个世界切换
  - 解开谜题到达终点
- **特色**: 水墨画风，禅意音乐

#### 11. 高原审问（Interrogate）
**地域特色**: 西藏布达拉宫
- **背景**: 有人试图破坏汉藏和睦
- **嫌疑人**: 3 个（僧人、商人、官员）
- **玩法**: 
  - 审问获取线索
  - 找出矛盾点
  - 指认真凶
- **特色**: 融入藏族文化、历史背景

#### 12. 荒诞科举（Absurd）
**地域特色**: 北京贡院
- **设定**: 一场荒诞的科举考试
- **考题**:
  - "如果孔子会武功，他能打几个梁山好汉？"
  - "请用微积分证明'己所不欲勿施于人'"
- **玩法**: 
  - 选择最合理的答案
  - 或者自己写答案（LLM 评分）
- **特色**: 黑色幽默，讽刺八股文

## 🎮 新增关卡类型详细设计

### 13. 华山论剑（Fighting Arena）
**地域特色**: 西岳华山
- **类型**: 横版格斗（类似拳皇）
- **玩法**: 
  - 1v1 擂台战
  - 使用中国武术（拳、掌、剑）
  - 气功系统（能量条）
- **对手**:
  - 东邪弟子（剑法）
  - 西毒传人（毒掌）
  - 南帝门人（一阳指）
  - 北丐徒弟（降龙十八掌）
  - 中神通（Boss，全真剑法）
- **特色**: 武侠风，招式华丽

### 14. 丝绸之路（Trade Route）
**地域特色**: 从长安到西域
- **类型**: 经营 + 冒险
- **玩法**:
  - 购买货物（丝绸、瓷器、茶叶）
  - 选择路线（沙漠、草原、雪山）
  - 遭遇事件（沙盗、暴风雪、迷路）
  - 交易获利
- **目标**: 到达西域（罗马），赚取 1000 两银子
- **特色**: 历史贸易路线，地理知识

### 15. 中医治病（TCM Healing）
**地域特色**: 各地医馆
- **类型**: 模拟 + 解谜
- **玩法**:
  - 诊断病人（望闻问切）
  - 选择药方（人参、黄芪、当归等）
  - 针灸穴位选择
- **病例**:
  - 风寒感冒（简单）
  - 脾胃虚弱（中等）
  - 疑难杂症（困难，李时珍都治不好）
- **特色**: 中医文化传播
