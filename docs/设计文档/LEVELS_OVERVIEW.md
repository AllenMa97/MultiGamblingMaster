## 🎮 Godot 关卡对应关系

### Speed Run 关卡脚本
`godot_client/scripts/levels/speed_run.gd`

**适配关卡**: 13 个
- 所有城市的"跑酷"类关卡
- 特点：快速移动、收集物品、计时挑战

### Castle Battle 关卡脚本
`godot_client/scripts/levels/castle_battle.gd`

**适配关卡**: 11 个
- 所有"攀登建筑"类关卡
- 特点：逐层挑战、击败敌人、到达塔顶

### Platform Challenge 关卡脚本
`godot_client/scripts/levels/platform_challenge.gd`

**适配关卡**: 5 个
- 所有"攀爬山峰"类关卡
- 特点：垂直向上、平台跳跃、精确操作

### Boss Battle 关卡脚本
`godot_client/scripts/levels/boss_battle.gd`

**适配关卡**: 2 个
- 所有"挑战神兽"类关卡
- 特点：单挑 Boss、躲避攻击、寻找弱点

## 📝 备注

1. **关卡数量**: 大部分城市 2 个关卡，符合设计要求
2. **难度递进**: 从江南→岭南→西南→华中，难度逐渐提升
3. **类型分布**: Speed Run 最多（适合快速穿梭），Boss Battle 最少（作为终极挑战）
4. **连锁剧情**: 需要完成特定城市的所有关卡才能触发
5. **Godot 复用**: 4 个基础脚本通过配置适配 31 个关卡
