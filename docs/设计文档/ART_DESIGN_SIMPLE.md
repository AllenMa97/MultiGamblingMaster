# 🎨 华夏游录 - 美术资源设计（简易版）

## 📦 角色设计：玉石精灵

### 主角：石头（玉石精灵）
**设计理念**：以中国各地玉石为原型的卡通形象

#### 基础形象
```
名称：石头（Stone）
原型：和田玉
颜色：温润白色带淡绿
性格：坚韧、温和、有智慧
特征：
  - 圆润的身体（像一块打磨过的玉石）
  - 两只小短手和小短脚
  - 大眼睛（黑色瞳孔，白色高光）
  - 头顶有一缕飘动的"玉气"（半透明绿色）
  - 微笑表情
```

#### 各地玉石变体（解锁皮肤）
1. **和田玉**（初始）- 温润白色
2. **翡翠**（云南）- 翠绿色
3. **岫岩玉**（东北）- 淡黄色
4. **蓝田玉**（陕西）- 淡蓝色
5. **独山玉**（河南）- 多彩色

### NPC 设计
1. **徐霞客**（人类向导）
   - 明代旅行家装扮
   - 手持折扇，背负行囊
   - 提供任务指引

2. **城市守护灵**（每个城市）
   - 当地文化元素的精灵
   - 苏州：园林精灵（亭台楼阁装饰）
   - 杭州：西湖精灵（荷花、断桥元素）
   - 广州：岭南精灵（木棉花、骑楼元素）
