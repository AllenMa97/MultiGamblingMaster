# 游戏概述

## 基本信息

- **游戏名称：** 华夏游录
- **游戏类型：** 中国风地图游历 + 横版动作冒险
- **游戏平台：** PC (Windows)
- **开发引擎：** Godot 4.6.2
- **后端框架：** FastAPI (Python)
- **游戏语言：** 简体中文

## 游戏特色

### 1. 完整的中国地图游历系统
- **65 座主要城市** - 覆盖全国 33 个省级行政区
- **195 个特色关卡** - 每个城市 3 个独特关卡
- **真实地理坐标** - 基于阿里云 DataV GeoJSON 数据
- **省界清晰标注** - 32 个省级行政区划分明确

### 2. 丰富的关卡类型
- **极速跑酷** - 考验操作与反应
- **城堡战斗** - 策略性战斗挑战
- **平台跳跃** - 精准跳跃挑战

### 3. 难度分布合理
- **五星级难度体系** - 符合正态分布
- 1 星（10%）- 新手友好
- 2 星（20%）- 入门挑战
- 3 星（40%）- 核心体验
- 4 星（20%）- 高难度挑战
- 5 星（10%）- 专家级挑战

# 背景故事

## 世界观

在古老的东方大陆上，流传着一个关于"山海经"的传说。传说中，华夏大地的每一座城市都蕴含着神秘的力量，每一片土地都承载着上古神话的记忆。

## 主角设定

**姓名：** 石头  
**身份：** 游历者  
**使命：** 游历华夏大地，体验山海经神话，收集城市之力

## 故事背景

上古时期，女娲补天遗留的五彩石散落在华夏大地，化为 65 座城市的守护灵。千年后，一位名叫"石头"的少年，意外获得了与这些守护灵沟通的能力。

为了揭开身世之谜，石头踏上了游历华夏大地的旅程。他需要：
- 游历 65 座城市
- 完成 195 个关卡挑战
- 收集城市守护灵的力量
- 揭开山海经的终极秘密

## 核心主题

- **探索** - 探索华夏大地的壮丽山河
- **成长** - 通过挑战不断变强
- **文化** - 体验中国传统文化的精髓
- **冒险** - 面对未知的挑战与机遇

# 核心玩法

## 1. 地图游历系统

### 游戏界面
```
┌────────────────────────────────────┐
│  中国地图 - 65 城 195 关            │
│  中华人民共和国                    │
│                                    │
│         [中国地图]                 │
│         (绿色国土)                 │
│         · 北京 (起点)              │
│         · 上海                     │
│         · 广州                     │
│         · 成都                     │
│         · 西安                     │
│                                    │
│  当前位置：北京                    │
│  城市总数：65 | 关卡总数：195      │
│                                    │
│  [掷骰子]  骰子：-                 │
└────────────────────────────────────┘
```

### 玩法规则
1. **选择起点** - 从北京出发
2. **掷骰子** - 随机 1-6 点
3. **移动** - 按点数移动到下一城市
4. **触发关卡** - 到达城市后自动触发
5. **完成挑战** - 通过关卡获得奖励
6. **继续旅程** - 重复上述步骤

## 2. 关卡挑战系统

### 关卡类型

#### 极速跑酷
- **目标** - 在最短时间内到达终点
- **挑战** - 避开障碍物，利用加速道具
- **技巧** - 掌握跳跃时机，选择最优路线

#### 城堡战斗
- **目标** - 击败所有敌人
- **挑战** - 敌人类型多样，需要策略应对
- **技巧** - 合理运用技能，观察敌人攻击模式

#### 平台跳跃
- **目标** - 到达最高点或最远点
- **挑战** - 精准跳跃，避免掉落
- **技巧** - 掌握跳跃力度和方向

### 关卡流程

```
进入关卡
  ↓
加载关卡场景
  ↓
显示关卡目标
  ↓
开始挑战
  ↓
完成/失败
  ↓
返回地图
  ↓
继续旅程
```

## 3. 难度系统

### 星级难度分布
```
难度星级 | 占比 | 关卡数量 | 描述
---------|------|----------|------------------
1 星     | 10%  | 19-20 个  | 新手教学关卡
2 星     | 20%  | 39 个     | 入门级挑战
3 星     | 40%  | 78 个     | 核心体验关卡
4 星     | 20%  | 39 个     | 高难度挑战
5 星     | 10%  | 19-20 个  | 专家级挑战
```

### 难度曲线
```
关卡数量
   ^
   |     ***
   |    *   *
   |   *     *
   |  *       *
   | *         *
   |*           *
   +--1-2-3-4-5--> 难度星级
      正态分布
```

# 游戏系统

## 1. 存档系统

### 存档数据结构
```json
{
  "save_id": "save_1234567890",
  "node_id": 0,
  "current_city": "北京",
  "completed_levels": [],
  "total_dice_rolls": 0,
  "start_time": 1234567890
}
```

### 存档位置
- **后端存储** - `backend/data/saves/{save_id}.json`
- **存档 ID 生成** - 基于时间戳的唯一 ID

## 2. API 系统

### 后端接口
- **基础 URL** - `http://localhost:8001`
- **API 版本** - v1
- **数据格式** - JSON

### 主要接口

#### 获取关卡配置
```
GET /api/action/level/{level_id}
Response: {
  "level_id": "beijing_run_001",
  "name": "北京极速跑酷",
  "difficulty": 3,
  "type": "speed_run"
}
```

#### 保存游戏进度
```
POST /api/game/save
Body: {
  "save_id": "save_123",
  "game_state": {...}
}
```

## 3. 场景系统

### 场景架构
```
Main.tscn (主场景)
  └─ MainMenu.tscn (主菜单)
      ├─ 标题
      ├─ 开始游戏按钮
      └─ 退出游戏按钮

MapScene.tscn (中国地图)
  ├─ 地图背景
  ├─ 玩家角色
  ├─ 城市标记
  ├─ UI 界面
  └─ 掷骰子按钮

[关卡场景]
  ├─ speed_run.tscn (极速跑酷)
  ├─ castle_battle.tscn (城堡战斗)
  └─ platform_challenge.tscn (平台跳跃)
```

## B. 关卡类型说明

- **核心玩法** - 快速到达终点
- **评分标准** - 用时越短分数越高

## C. 常见问题解答

**Q: 游戏支持多人联机吗？**  
A: 当前版本为单人游戏，多人模式正在开发中。

**Q: 可以跳过某个关卡吗？**  
A: 可以，但会影响最终评分和成就解锁。

**Q: 存档可以转移吗？**  
A: 可以，存档文件位于 `backend/data/saves/` 目录。

**Q: 游戏有中文配音吗？**  
A: 目前只有背景音乐和音效，暂无配音计划。
