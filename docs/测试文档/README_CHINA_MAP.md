## 📖 游戏简介

一个基于 Web 的中国风游历冒险游戏，玩家扮演明代旅行家徐霞客，游历中华大地，体验各地文化。结合传统副本与 AI 驱动的互动玩法，使用 FastAPI 后端、Phaser 前端和 Godot 横版动作引擎打造。

### 🎯 游戏特色

- **中国风游历**：扮演徐霞客，游览中华大地
  - 🏯 江南水乡 - 苏州园林、杭州西湖
  - 🌸 岭南风情 - 广州、深圳、桂林
  - ⛰️ 西南秘境 - 成都、重庆、昆明、大理
  - 🏛️ 华中腹地 - 武汉、长沙、张家界

- **17 个特色城市**：
  - 每个城市都有 2-3 个专属 Godot 横版动作挑战
  - 难度从★到★★★★★，适合不同水平的玩家
  - 城市间有解锁关系，需要逐步探索

- **40+ 精彩挑战**：
  - **Godot 横版动作**（100%）:
    - 🏃 园林跑酷 - 在亭台楼阁间穿梭
    - ⚔️ 古塔战斗 - 攀登雷峰塔、武侯祠
    - 🧗 登山攀岩 - 黄山、华山、黄果树瀑布
    - 🐉 神兽 Boss - 挑战各地传说神兽

- **连锁剧情系统**：
  - 完成特定城市和挑战组合触发特殊剧情
  - "白蛇传"：在杭州完成雷峰塔和西湖挑战
  - "蜀道难"：完成成都和重庆的所有挑战

- **双引擎架构**：Phaser（地图探索）+ Godot（横版动作），无缝切换
- **存档系统**：完整的游戏进度保存和读取功能
- **精美 UI**：中国风水墨画风，传统音乐配乐

## 🎮 快速开始

### 方式 1：使用启动脚本（推荐）

```bash
# 安装依赖（首次运行）
npm install

# 启动游戏
node start.js
```

### 方式 2：手动启动

#### 1. 启动后端
```bash
cd backend
pip install -r requirements.txt
python main.py
```
后端将在 `http://localhost:8001` 启动

#### 2. 启动前端
```bash
# 在新终端窗口
cd frontend
npm install
npm run dev
```
前端将在 `http://localhost:5173` 启动

#### 3. 打开浏览器
访问 `http://localhost:5173` 开始游戏！

## 🎯 游戏玩法

### 基本流程

1. **选择起点**：游戏开始时选择起点城市（苏州/广州/成都/武汉）
2. **探索城市**：点击蓝色城市前往游历
3. **接受挑战**：到达城市后选择挑战
4. **完成挑战**：进入 Godot 横版动作关卡
5. **解锁新城市**：完成挑战后解锁相邻城市
6. **触发剧情**：完成特定组合触发连锁剧情
7. **游历全国**：探索所有 17 个城市

### 颜色系统

- 🔴 **红色城市**：当前所在城市（点击选择挑战）
- 🔵 **蓝色城市**：可前往的城市（点击移动）
- 🟡 **金色城市**：已访问过的城市
- ⚪ **灰色城市**：尚未解锁（需要先解锁相邻城市）

### 移动规则

- 只能前往已解锁的城市（蓝色）
- 城市之间有解锁关系（完成一个城市后解锁相邻城市）
- 不能跳跃式移动，需要逐步探索
